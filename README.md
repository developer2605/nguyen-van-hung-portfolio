# Portfolio Nguyễn Văn Hùng

Website tĩnh gồm HTML/CSS/JavaScript, có thể mở trực tiếp hoặc triển khai trên GitHub Pages, Netlify, Vercel.

## Xem trên máy tính

Mở file `index.html` bằng trình duyệt.

## Cấu trúc

- `index.html`: nội dung website
- `styles.css`: giao diện responsive
- `script.js`: menu mobile, hiệu ứng cuộn, bộ đếm
- `assets/nguyen-van-hung.jpg`: ảnh chân dung
- `assets/Nguyen-Van-Hung-CV.pdf`: CV tải về

## Triển khai nhanh

1. Kéo toàn bộ thư mục vào Netlify Drop; hoặc
2. Đưa lên repository GitHub và bật GitHub Pages; hoặc
3. Import repository vào Vercel.

## Nội dung nên bổ sung ở phiên bản tiếp theo

- Logo chính thức dạng PNG/SVG của Bệnh viện Phương Đông và LEDA Clinic.
- Screenshot Ads Manager, dashboard, creative hiệu quả.
- Chứng chỉ và ảnh giải thưởng.
- Tên miền cá nhân và form liên hệ gửi dữ liệu thật.
